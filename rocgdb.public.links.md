# introduction to ROCgdb
+ [This is ROCgdb, the ROCm source-level debugger for Linux, based on GDB, the GNU source-level debugger.](https://github.com/ROCm/ROCgdb)
    + [ROCgdb documentation](https://rocm.docs.amd.com/projects/ROCgdb/en/latest/)
+ [The AMD Debugger API is a library that provides all the support necessary for a debugger and other tools to perform low level control of the execution and inspection of execution state of AMD's commercially available GPU architectures.](https://github.com/ROCm/ROCdbgapi)
    + [AMD Debugger API (ROCdbgapi)](https://github.com/ROCm/ROCdbgapi)
+ [TheRock (The HIP Environment and ROCm Kit) is a lightweight open source build platform for HIP and ROCm.](https://github.com/ROCm/TheRock)
+ [ROCgdb exercises tested on MI210 and MI300A](https://github.com/amd/HPCTrainingExamples/tree/main/Rocgdb)
+ [HPCTrainingDock](https://github.com/amd/HPCTrainingDock)
